# Digit Memory Trainer

A modern desktop application designed to improve your digit memory skills through progressive challenges. Test and train your ability to memorize sequences of numbers from 3 to 15 digits across multiple difficulty levels.

## Features

- **13 Progressive Levels**: Start with 3-digit numbers and work your way up to 15-digit challenges
- **Adaptive Difficulty**: Each level increases in complexity with more digits and shorter memorization times
- **Modern UI**: Vibrant, engaging interface with smooth animations and responsive design
- **Score Tracking**: Detailed scoring system based on accuracy and speed
- **Achievements System**: Unlock achievements for milestones like perfect scores, speed records, and completion
- **Progress Persistence**: Save your progress locally and track high scores per level
- **Gamification Elements**: Visual feedback, level unlocks, and rewarding animations

## Technology Stack

- **Electron**: Cross-platform desktop app framework
- **React**: Component-based UI library
- **TypeScript**: Type-safe JavaScript for better development experience
- **Tailwind CSS**: Utility-first CSS framework for modern styling

## Installation

1. Clone the repository and navigate to the project directory.
2. Run `npm install` to install all dependencies.
3. For development: Run `npm run dev` to start the development servers for both Electron and React.
4. For production: Run `npm run build` to create a production build, then `npm start` to launch the app.

## How to Play

1. **Select a Level**: Choose from 13 levels ranging from 3 to 15 digits. Levels are unlocked progressively as you complete them.
2. **Memorize the Number**: A random number will be displayed for a limited time (based on level difficulty). Study it carefully!
3. **Input Phase**: After the number disappears, enter the digits you remember into the input field.
4. **Get Scored**: Your accuracy and speed determine your score. Perfect recalls and fast times earn higher points.
5. **Progress**: Complete levels to unlock the next ones and earn achievements.

**Tips**: Use chunking (grouping digits) and visualization techniques to improve your memory. Practice regularly to see improvement!
